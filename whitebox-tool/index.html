Tips

  • Untuk operator && atau ||, tambahkan +1 Complexity per operator.
  • Contoh: if ($a && $b) → Complexity = 2
  • Node ID harus unik (angka positif).
  • Setiap graph harus punya minimal 1 node entry dan 1 node exit.

💡 Petunjuk Template:

  • Gunakan page.goto('/') untuk navigasi.
  • Akhiri dengan asersi expect(...).
// === KODE BOILERPLATE (Otomatis) ===
test('AK-F-001: Validasi Field Wajib', async ({ page }) => {
  await loginAs(page, 'admin');
  await page.goto('/kegiatan/create');


  await expect(page.locator('text=Penanggung Jawab wajib diisi')).toBeVisible();
});

// turbo-all
1.  **Read Error**: Analyze the error message and stack trace provided.
2.  **Locate**: Find the file and line number mentioned.
3.  **Identify Cause**: Determine the root cause (syntax error, missing import, type mismatch, etc.).
4.  **Apply Fix**: correct the code.
5.  **Verify**: Run `composer test` to ensure no regressions.
6.  **Explain**: Briefly explain what caused the error and how it was fixed.
